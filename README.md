# Event Guest List Manager

This is a simple JavaScript project that allows users to manage a guest list for an event.
## Features

- Add a guest name through a form
- Display the guest name in a numbered list
- Remove individual guests using a "Remove" button
- Limit the guest list to 10 entries
- Show an alert if the name field is empty or the list limit is reached

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla)


It is focused on basic DOM manipulation and JavaScript event handling.
